# 《神明公關：誰是蹟優股》桌遊募資宣傳網頁

本專案為國立聯合大學專題原創策略桌遊——《神明公關：誰是蹟優股》的募資與數位導入先行版宣傳網頁。本資料夾為**「新版本完整網站區」**，所有最新代碼與重命名後的實體美術素材皆收錄於此。

## 📈 重大改動與版本演進紀錄

只有發生「重大架喚變更、核心功能新增、或實體素材遷移替換」時才會進行版本更新與紀錄：

### ✦ v3.4.0 (最新版本) — 2026-06-11
*   **【立繪優化】還原神使自然色彩**：移除蘇映安與周宛辭立繪上的偏色 `hue-rotate` 濾鏡，全面清除 character card 圖像之飽和度降減設置，還原立繪自然精美色彩，並調整 hover 時之亮度與對比。
*   **【排版優化】寬版排版大氣延展**：將網站最大容器寬度 `.container` 及導航、Hero Content 的最大寬度限制由 `1160px` 擴展至 `1400px`，內邊距調寬至 `40px`。同時將角色卡、卡牌展示與贊助方案之格線間距 `gap` 分別調大至 `32px` / `28px`，使寬螢幕下視覺配置更為大氣。
*   **【排版優化】標題字級層次微調**：調小 H1 首頁標題文字，突顯標準字 Logo (`logo.png`) 的第一視覺焦點，並藉由土金色與漸層金黃的主副標層次，重塑優質宮廟桌遊宣傳排版。
*   **【互動優化】平滑滾動至過渡交接處**：重構 `scrollToMainContent()` 滾動函數。點擊「滾動進入遊戲」與開場 3 秒之自動滑動，皆升級為 smooth 平滑滾動，且精準停止於 `#hero` 上方留白 160 像素的交接重疊處，保留第零頁大圖底部，使區塊過渡更加溫和滑順。
